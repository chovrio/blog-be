# 后台

## 项目简介

个人博客项目后台

## 运行

```
  npm i -g pnpm # 如果已有安装可跳过
  pnpm i
  pnpm dev
```

### 打包

```
  pnpm build
  pnpm preview # 预览
```

## 项目依赖

- react
- typescript
- antd
- axios
- react-router-dom
- @wangeditor(富文本编辑器)
- react-markdown
- redux-tooltik

### 功能

- 系统总览
- 用户管理
- 文章管理
